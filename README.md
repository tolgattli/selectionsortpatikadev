## [22,27,16,2,18,6] -> Insertion Sort

```

* [2,27,16,22,18,6] (n) 
* [2,27,16,22,18,6] (n-1)
* [2,6,16,22,18,27] (n-2)
* [2,6,16,18,22,27] (1)  *//Sorting completed*

```

## Big-O Notation


```
>* n+(n-1)+(n-2)+...+1= n(n+1) /2
>* Big-O = (n^2)

```
